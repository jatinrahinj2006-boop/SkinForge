# SkinForge

Convert Minecraft skins to Litematica format with an intuitive web-based interface.

## Features

- **Skin Upload**: Load and preview your Minecraft skins
- **3D Visualization**: View your skin as a 3D statue in real-time
- **Customizable Conversion**: Adjust settings to match your needs
- **Litematica Export**: Generate schematic files compatible with the Litematica mod
- **Responsive Design**: Works on desktop and mobile browsers
- **Dark Theme**: Minecraft-inspired UI with a retro 8-bit aesthetic

## How to Use

1. Open `SkinForge_v2.html` in your web browser
2. Upload a Minecraft skin file (PNG format)
3. Preview the skin and 3D model
4. Adjust conversion settings as needed
5. Click "Generate" to create your Litematica schematic
6. Download the `.litematica` file

## Requirements

- Modern web browser with JavaScript enabled
- Minecraft skin file (64x64 PNG recommended)
- Litematica mod installed in Minecraft (for importing schematics)

## Technologies

- HTML5
- CSS3 with CSS Variables
- Vanilla JavaScript
- Canvas API for 2D and 3D rendering
